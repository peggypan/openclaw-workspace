# GitHub Trending Monitor Configuration

## 功能
自动监测 GitHub 每日 Star 增长最快的项目

## 数据来源
- URL: https://github.com/trending?since=daily
- 抓取方式: Camoufox 浏览器
- 更新频率: 每日 (通过 Heartbeat)

## 历史数据
数据保存在 `memory/github-trending-YYYY-MM-DD.md` 文件中
