# 2026-03-09 - 雅思备考规划

## 用户背景
- **当前水平：** 英语六级（有一段时间没接触）
- **可用时间：** 每天 1 小时
- **目标分数：** 雅思 7.5（6个月）
- **备考方式：** 自学 + AI教练
- **弱项：** 四项都需要加强

## 每日时间分配建议（60分钟）
| 时间段 | 内容 | 时长 |
|--------|------|------|
| 工作日 | 听力/阅读/写作/口语 轮训 | 40-45 min |
| | 词汇积累 | 15-20 min |
| 周末 | 完整模考或专项突破 | 灵活安排 |

## 6个月阶段规划

### 第一阶段：唤醒期（第1-2月）
- **目标：** 恢复英语感觉，熟悉雅思题型
- **听力：** 剑雅真题 Section 1-2 精听
- **阅读：** 掌握定位技巧，限时练习
- **写作：** 学习 Task 1/2 结构，积累句型
- **口语：** 准备 Part 1 话题，录音练习
- **词汇：** 雅思核心词汇 3000（每天 50 个）

### 第二阶段：突破期（第3-4月）
- **目标：** 分项强化，攻克难点
- **听力：** Section 3-4 学术场景，同义替换训练
- **阅读：** 长难句分析，True/False/Not Given 技巧
- **写作：** 高分范文仿写，观点积累
- **口语：** Part 2 串题，Part 3 深度回答
- **词汇：** 同义替换词库，话题词汇

### 第三阶段：冲刺期（第5-6月）
- **目标：** 模考冲刺，查漏补缺
- 每周 1-2 次完整模考
- 针对弱项最后突破
- 应试技巧强化

## 互动方式
- **每日任务：** 晚上推送次日学习计划
- **练习反馈：** 用户可发送语音/文字，我纠正发音和语法
- **定期检测：** 每两周一次阶段性测试
