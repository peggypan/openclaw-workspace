### 📝 如何使用子代理

子代理通过 OpenClaw 的 `sessions_spawn` 功能调用：

```python
# 示例：调用股票专家分析某只股票
sessions_spawn(
    task="请分析腾讯控股(00700)的投资价值，包括基本面、行业地位和估值水平",
    agentId="stock-expert"
)

# 示例：让产品经理帮你规划功能
sessions_spawn(
    task="我想做一个记账App，请帮我做需求分析和产品规划",
    agentId="product-manager"
)
```

### 🔄 未来扩展计划

可以进一步为每个子代理添加：
- **MEMORY.md** - 长期记忆（投资记录、项目历史等）
- **HEARTBEAT.md** - 定时任务（每日市场简报、项目进度提醒等）

### 💡 使用建议

1. **专业分工** - 不同领域的问题找对应的专家
2. **组合使用** - 复杂项目可以多个子代理协作
3. **记忆积累** - 鼓励子代理记录重要信息到 MEMORY.md
4. **定期复盘** - 让子代理定期总结工作成果
