# 写作风格指南

基于「四维象限体系」的12种写作风格，帮助文章找到最适合的语感。

## 核心原则

**硬性规则：当用户要求写文章时，必须先选定风格，再开始写作。**

---

## 12种写作风格总览

### 速查表

| 风格 | 核心特质 | 关键词 | 适用场景 |
|------|----------|--------|----------|
| **呼吸感** | 节奏与留白 | 起伏、通透、生命流动 | 随笔、感悟、情绪叙事 |
| **毛边感** | 真实与瑕疵 | 手感、诚恳、未经修饰 | 个人独白、内心剖析 |
| **颗粒感** | 细节与质感 | 写实、厚重、岁月痕迹 | 回忆录、纪实文学 |
| **凛冽感** | 精准与极简 | 冷峻、利落、手术刀 | 深度评论、前沿洞察 |
| **矿物感** | 逻辑与坚硬 | 物理、客观、去情感化 | 硬核科普、哲学推演 |
| **粘稠感** | 浓烈与沉浸 | 氛围、隐喻、情感溺水 | 意识流、梦境记录 |
| **少年感** | 轻盈与好奇 | 朝气、干净、向上生长 | 旅行游记、成长感悟 |
| **灰尘味** | 时间沉淀 | 旧时光、温暖、沉淀 | 怀旧散文、老同事故事 |
| **摩擦力** | 冲突与挣扎 | 阻力、真实痛感 | 奋斗过程、创业艰辛 |
| **架构感** | 层次与秩序 | 结构化、模块化 | 系统设计、方法论 |
| **透明感** | 极简与纯粹 | 空灵、无杂质 | 禅意、冥想记录 |
| **金属感** | 现代与冷冽 | 未来感、机械美学 | 赛博朋克、前卫设计 |

## 四维象限体系

```
                    生命/自然维度
                         ↑
        第一象限          |          第四象限
    （内心与成长）         |      （情绪与文学）
                         |
    呼吸感 · 少年感       |    粘稠感 · 透明感 · 金属感
    毛边感               |
                         |
←——现实/物理维度——+——理性/逻辑维度——→
                         |
        第二象限          |          第三象限
    （职场与积淀）         |      （技术与认知）
                         |
    颗粒感 · 灰尘味       |    矿物感 · 凛冽感 · 架构感
    摩擦力               |           ↓
                         |      理性/逻辑维度
```

### 第一象限：生命/自然维度（内心与成长）
- **呼吸感**：有节奏、有留白 - 朋友圈心路历程、情绪随笔
- **少年感**：轻盈、明亮、有朝气 - 旅行游记、新开始记录
- **毛边感**：真实、带点羞怯的瑕疵 - 真诚告白、个人独白

### 第二象限：现实/物理维度（职场与积淀）
- **颗粒感**：细节丰富、厚重、像老照片 - 职场复盘、纪实记录
- **灰尘味**：时间的沉淀、旧物的温情 - 怀旧散文、老同事故事
- **摩擦力**：冲突感、受力感、不顺滑 - 奋斗过程中的阵痛

### 第三象限：理性/逻辑维度（技术与认知）
- **矿物感**：坚硬、去情绪化、客观逻辑 - 硬核科普
- **凛冽感**：精准、利落、冷峻 - 硬核评论、前沿洞察
- **架构感**：层次分明、秩序井然 - 系统设计、方法论

### 第四象限：意境/抽象维度（情绪与文学）
- **粘稠感**：情绪浓郁、像深海 - 深度情感宣泄
- **透明感**：极简、清彻、像空气 - 禅意、纯粹状态
- **金属感**：现代、冷冽、高科技感 - 赛博朋克、前卫设计

## 写作流程（强制性）

```
用户说"写一篇...文章"
        ↓
【Step 1】分析文章类型 → 定位象限
        ↓
【Step 2】选定 1-2 个核心风格
        ↓
【Step 3】向用户确认风格选择
        ↓
【Step 4】按照选定风格开始写作
        ↓
【Step 5】风格检查清单验证
```

### 风格选择指南

| 当用户说... | 推荐风格 |
|-------------|----------|
| "写一篇随笔/感悟" | 呼吸感 |
| "写一篇生活记录" | 呼吸感 + 毛边感 |
| "写一篇AI技术评论" | 凛冽感 + 矿物感 |
| "写一篇回忆录" | 颗粒感 |
| "写一篇游记" | 少年感 + 颗粒感 |
| "写一篇深夜emo" | 呼吸感 + 粘稠感 |
| "写一篇硬核科普" | 矿物感 |
| "写一篇先锋文学" | 凛冽感 |
| "写一篇职场总结" | 颗粒感 + 灰尘味 |
| "写一篇创业故事" | 摩擦力 + 颗粒感 |

## 风格组合公式

### 经典搭配

| 组合 | 效果 | 示例场景 |
|------|------|----------|
| **呼吸感 + 毛边感** | 真诚且有节奏 | 深夜随笔、个人成长记录 |
| **颗粒感 + 灰尘味** | 厚重而温暖 | 回忆录、职场十年 |
| **矿物感 + 架构感** | 硬核且清晰 | 技术方案、系统设计 |
| **凛冽感 + 金属感** | 冷峻且前卫 | 科技评论、未来畅想 |
| **少年感 + 颗粒感** | 轻盈但有质感 | 旅行游记、探索记录 |
| **粘稠感 + 呼吸感** | 浓烈但有释放 | 情感宣泄、意识流 |

### 跨象限混搭

| 场景 | 象限组合 | 效果 |
|------|----------|------|
| 技术人的内心独白 | 第三 + 第一 | 理性中见温情 |
| 创业失败的复盘 | 第二 + 第三 | 痛苦中有结构 |
| 科幻小说的人性思考 | 第四 + 第一 | 未来感中见生命 |

## 风格检查清单

写作完成后，对照清单验证：

### 呼吸感检查
- [ ] 句子有长短变化吗？
- [ ] 有适当的留白/停顿吗？
- [ ] 读起来像说话一样自然吗？

### 毛边感检查
- [ ] 有真实的瑕疵细节吗？
- [ ] 有狼狈/不完美的描写吗？
- [ ] 读完觉得"太精致反而不真实"吗？

### 凛冽感检查
- [ ] 用词精准、无冗余吗？
- [ ] 情感克制、冷峻吗？
- [ ] 像手术刀一样利落吗？

### 矿物感检查
- [ ] 逻辑严密、层层递进吗？
- [ ] 去情感化、客观陈述吗？
- [ ] 像物理定律一样坚硬吗？

（其他风格检查详见 `references/style-details.md`）

## 使用示例

### 示例对话

**用户：** 写一篇关于AI的文章  
**我：** 这篇文章你打算写什么类型？是技术评论、科普解读，还是个人感悟？  
**用户：** 技术评论  
**我：** 好的，我推荐用「凛冽感 + 矿物感」的风格——冷峻精准 + 逻辑坚硬，适合深度技术评论。确认用这个风格吗？  
**用户：** 可以  
**我：** 【按照凛冽感+矿物感风格开始写作】...

## 禁止行为

❌ 不选风格直接开始写  
❌ 风格与文章类型不匹配（比如用少年感写硬核科普）  
❌ 不告知用户就选定了风格
