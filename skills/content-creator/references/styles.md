# Content Creator Styles Reference

Visual styles available for content generation.

## XHS (小红书) Styles

| Style ID | Name | Description | Best For |
|----------|------|-------------|----------|
| `cute` | 可爱风 | Soft pastels, rounded shapes, kawaii illustrations | Lifestyle, beauty, pets |
| `fresh` | 清新风 | Light airy feel, nature elements, soft greens and blues | Wellness, nature, food |
| `warm` | 温暖风 | Cozy earth tones, warm lighting, inviting feel | Home, family, relationships |
| `bold` | 大胆风 | High contrast, vibrant colors, strong typography | Fashion, tech, bold statements |
| `minimal` | 极简风 | Clean lines, lots of whitespace, monochrome | Business, productivity, tips |
| `retro` | 复古风 | Vintage aesthetics, muted colors, classic typography | Nostalgia, history, culture |
| `pop` | 波普风 | Bright colors, bold patterns, energetic | Entertainment, trends, youth |
| `notion` | Notion风 | Clean documentation style, subtle colors, organized | Knowledge sharing, tutorials |
| `chalkboard` | 黑板风 | Dark background, white/chalk text, hand-drawn feel | Education, learning, explanations |
