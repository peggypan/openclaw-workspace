# Content Creator Layouts Reference

Available layout structures for infographic generation.

## Infographic Layouts

### Process & Flow

| Layout ID | Name | Description | Use Case |
|-----------|------|-------------|----------|
| `bridge` | Bridge | Problem → Solution visualization | Showing transformation or solution |
| `circular-flow` | Circular Flow | Cyclical process diagram | Recurring processes, cycles |
| `flow` | Flow (XHS) | 3-6 step process flow | Step-by-step tutorials |
| `timeline-horizontal` | Timeline | Chronological events | History, roadmaps |
| `journey-path` | Journey Path | Winding path with milestones | User journey, career paths |

### Comparison & Analysis

| Layout ID | Name | Description | Use Case |
|-----------|------|-------------|----------|
| `comparison-table` | Comparison Table | Side-by-side comparison | Product comparisons |
| `comparison` | Comparison (XHS) | Two-column layout | Pros/cons, alternatives |
| `do-dont` | Do vs Don't | Correct vs incorrect practices | Guidelines, best practices |
| `scale-balance` | Scale Balance | Pros vs cons weighing | Decision making |
| `venn` | Venn | Overlapping concepts | Relationships, intersections |

### Hierarchy & Structure

| Layout ID | Name | Description | Use Case |
|-----------|------|-------------|----------|
| `pyramid` | Pyramid | Stacked hierarchy levels | Maslow's needs, priorities |
| `tree-hierarchy` | Tree Hierarchy | Branching structure | Org charts, taxonomy |
| `nested-circles` | Nested Circles | Concentric circles | Scope, influence levels |
| `layers-stack` | Layers Stack | Stacked layers | Tech stacks, architecture |
| `iceberg` | Iceberg | Visible vs hidden parts | Surface vs deep analysis |

### Information Display

| Layout ID | Name | Description | Use Case |
|-----------|------|-------------|----------|
| `feature-list` | Feature List | Bullet points with icons | Product features |
| `grid-cards` | Grid Cards | Card grid layout | Multi-topic overview |
| `list` | List (XHS) | Numbered or bulleted list | Rankings, collections |
| `priority-quadrants` | Priority Quadrants | 2x2 matrix grid | Prioritization frameworks |
| `equation` | Equation | Formula breakdown | Concepts, formulas |
| `fishbone` | Fishbone | Root cause diagram | Problem analysis |
| `funnel` | Funnel | Inverted pyramid | Conversion, filtering |
| `mind-map` | Mind Map | Radial expansion | Brainstorming, ideas |
| `sparse` | Sparse (XHS) | 1-2 points, lots of space | Cover, key quotes |
| `balanced` | Balanced (XHS) | 3-4 points, comfortable | Regular content |
| `dense` | Dense (XHS) | 5-8 points, information-rich | Knowledge cards |

## Layout Selection Guide

**For XHS (小红书) Images:**
- Cover page → `sparse`
- Tips/Lessons → `balanced` or `list`
- Knowledge sharing → `dense`
- Comparison content → `comparison`
- Tutorial/Steps → `flow`

**For Professional Infographics:**
- Process explanation → `flow`, `timeline`, `journey-path`
- Data comparison → `comparison-table`, `venn`, `scale-balance`
- Hierarchy/Levels → `pyramid`, `tree-hierarchy`, `layers-stack`
- Problem analysis → `fishbone`, `iceberg`, `bridge`
- Feature showcase → `feature-list`, `grid-cards`
- Decision framework → `priority-quadrants`, `do-dont`
